# Auditoria de Qualidade: Insider Nextech T-Shirt

## Resumo do Projeto
Este repositório contém a análise técnica de qualidade realizada sobre a NEXTECH™ T-Shirt da fabricante Insider. O trabalho foi desenvolvido como parte integrante do curso de Qualidade de Software da EBAC, aplicando metodologias de auditoria qualitativa e verificação de requisitos de durabilidade e performance.

## Especificações Técnicas
* **Produto:** NEXTECH™ T-Shirt Masculina
* **Composição:** 92% Liocel e 8% Elastano
* **Metodologia:** Inspeção técnica direta e análise de engenharia têxtil

## Matriz de Análise Qualitativa

| Dimensão | Percepção do Auditor | Evidência Técnica |
| :--- | :--- | :--- |
| **Matéria-prima** | Fibra de Liocel com toque macio | Composição de 92% Liocel |
| **Performance** | Gerenciamento térmico e anti-odor | Propriedades das fibras celulósicas |
| **Usabilidade** | Alta praticidade (não amassa) | Atributo anti-wrinkle verificado |
| **Design** | Minimalismo funcional | Costuras de perfil baixo e gola premium |
| **Durabilidade** | Alta resistência mecânica | Estabilidade dimensional após lavagem |

## Conclusões da Auditoria
A análise técnica validou que a alta performance da peça é resultado direto da seleção de matérias-primas nobres. O uso do Liocel garante uma resistência superior à do algodão tradicional, enquanto o elastano assegura a memória elástica do produto sob tensão.
